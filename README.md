# README

## Usage

The script requires you to manually set parameters before launching and also set up your Gemini API key in `.env` (use `.env.example` as template).

Among the parameters to modify, most important is the `systemPrompt` that describes what you're trying to achieve. A good default is already in-place for you that works for EvilGinx as an example (but you can run it on any repo with the prompt being as it is). Make sure to modify at least parts of it so that it fits your use-case.
